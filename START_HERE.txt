🚀 QUICK START (5 MINUTES):

Step 1: Start Database
  docker-compose up -d

Step 2: Setup Backend
  cd backend
  npm install
  cp .env.example .env
  npx prisma generate
  npx prisma migrate dev --name init
  npx prisma db seed
  npm run start:dev

Step 3: Setup Frontend (in new terminal)
  cd frontend
  npm install
  cp .env.example .env
  npm run dev

Step 4: Test It!
  Open: http://localhost:5173
  Login with: admin@modak-cafe.com / Admin@123
